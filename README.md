# Maartens Place - GitHub Pages PWA

Upload deze bestanden naar de root van je GitHub-repository.

Mapstructuur:

```text
/
├── index.html
├── manifest.json
├── service-worker.js
└── icons/
    ├── favicon-32.png
    ├── icon-192.png
    ├── icon-512.png
    └── logo-original.png
```

Na uploaden open je de GitHub Pages-link. In Chrome/Edge kun je de site daarna installeren via het icoon in de adresbalk of via het menu: Apps / Deze site installeren.
